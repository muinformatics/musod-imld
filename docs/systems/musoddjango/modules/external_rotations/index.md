### External Rotations Overview/Brief
- The ExternalRotations module is MUSOD's operational capture system for procedures completed in external clinic settings. It standardizes how student procedure activity is logged, grouped, reviewed, exported, and governed for downstream institutional use.support.
- Source of Truth: [Repo Muinformatics/MusodDjango](https://github.com/muinformatics/MusodDjango)
