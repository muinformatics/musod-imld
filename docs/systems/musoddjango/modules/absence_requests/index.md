### Absence Requests Management Overview/Brief
- The AbsenceRequests module is MUSOD's workflow engine for student absence governance. It standardizes how absence requests are created, validated, reviewed, approved/denied, archived, and reported.
- Source of Truth: [Repo Muinformatics/MusodDjango](https://github.com/muinformatics/MusodDjango)
